INSTALLATION
────────────
1. npx create-next-app@latest mon-site --typescript --tailwind --app
2. cd mon-site
3. npm install stripe
4. Copie les fichiers dans la bonne structure
5. Remplis .env.local avec tes vraies clés

STRIPE SETUP (15 min)
─────────────────────
1. Crée compte sur stripe.com
2. Dashboard → Produits → + Créer un produit
   Nom : "Formation Google UAC"
   Prix : 67€ · Paiement unique
   → Copie le Price ID (price_xxx) dans LandingClient.tsx ligne 32
3. Dashboard → Développeurs → Clés API
   → Copie Secret Key dans .env.local
4. Dashboard → Développeurs → Webhooks
   → Ajouter endpoint : https://ton-domaine.fr/api/webhook
   → Événement : checkout.session.completed
   → Copie Webhook Secret dans .env.local

ZAPIER SETUP (20 min)
─────────────────────
Zap 1 : Stripe → Gumroad (accès formation)
  Trigger : Stripe - New Successful Charge
  Action : Gumroad - Create Sale

Zap 2 : Stripe → Brevo (séquence email)
  Trigger : Stripe - New Successful Charge  
  Action : Brevo - Add Contact to List "Acheteurs"

Zap 3 : Stripe → Google Sheets (tracking)
  Trigger : Stripe - New Successful Charge
  Action : Google Sheets - Create Row
  Colonnes : Date / Email / Montant / Session ID

VERCEL DEPLOY (5 min)
─────────────────────
1. Push ton code sur GitHub
2. vercel.com → Import Project → ton repo
3. Settings → Environment Variables → colle ton .env.local
4. Deploy → ton site est en ligne

DOMAINE OVH (10 min)
─────────────────────
1. ovhcloud.com → cherche ton-domaine.fr → commande (~10€/an)
2. Dans Vercel → Settings → Domains → ajoute ton domaine
3. Vercel te donne les DNS → copie dans OVH
4. Propagation 1 à 24h
